Проект для `STM32CubeIDE`, рассчитанный на микроконтроллер `STM32F767`.

В репозитории находится структура проекта, созданная в среде STM32CubeIDE / STM32CubeMX. Проект можно открыть в IDE, собрать и использовать для дальнейшей разработки или проверки исходного кода.

Основные части проекта:

- `Core/` - исходный код приложения, включая `main.c`, обработчики прерываний и системные файлы STM32.
- `Drivers/` - драйверы HAL и файлы CMSIS, необходимые для работы с микроконтроллером.
- `demonstration.ioc` - файл конфигурации CubeMX, в котором хранятся настройки периферии, тактирования и проекта.
- `STM32F767ZITX_FLASH.ld` и `STM32F767ZITX_RAM.ld` - linker scripts для размещения программы в памяти микроконтроллера.
- `.project`, `.cproject`, `.mxproject`, `.settings/` - служебные файлы STM32CubeIDE / Eclipse, необходимые для корректного открытия проекта в среде разработки.
- `demonstration Debug.launch` - конфигурация запуска и отладки.

Папка `Debug/` содержит результаты сборки проекта: объектные файлы, dependency-файлы, `*.elf`, `*.map`, `*.list` и другие промежуточные данные. Эти файлы не являются исходным кодом и могут быть пересозданы IDE при следующей сборке, поэтому они исключены из репозитория через `.gitignore`.

Таким образом, в git хранится исходная и конфигурационная часть проекта, а автоматически создаваемые файлы сборки не добавляются.
